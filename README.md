JS5 - ვებ-გვერდის დავალება: დინამიური ელემენტები  

ეს არის პატარა ვებ-პროექტი, რომელიც შექმნილია იმის საჩვენებლად, თუ როგორ მუშაობს JavaScript-ი (JS) და CSS-ი ერთად, რათა ვებ-გვერდი გახდეს ინტერაქტიული და ლამაზი.

რა შეუძლია ამ პროექტს?

პროექტი შედგება რამდენიმე დამოუკიდებელი ნაწილისგან, რომლებიც გვიჩვენებს, როგორ ვმართოთ ვებ-გვერდი   JS-ის საშუალებით:

ჯადოსნური ღილაკი (CSS ანიმაცია):

თქვენ ნახავთ ღილაკს, რომელიც სტილიზებულია, როგორც ნეონის განათება და მუდმივად მოძრაობს ეკრანზე გვერდიდან გვერდზე.

როდესაც მას დააჭერთ, JavaScript უბრალოდ წაშლის ამ ღილაკს ეკრანიდან.

  სურათი:

JS კოდი   ქმნის <img> ტეგს და სვამს მას გვერდზე, რათა აჩვენოს, როგორ ხდება ელემენტების დინამიურად დამატება.

  გალერეის ბარათები (Grid Layout):

src/js/results.js ფაილში მოცემული მონაცემები (ნახატების შესახებ) ავტომატურად გარდაიქმნება ლამაზ ბარათებად.

CSS ხრიკი: ბარათები განლაგებულია 3-სვეტიან ბადეში (Grid), და ყველა სურათს აქვს ზუსტად ერთი და იგივე სიმაღლე, მიუხედავად ორიგინალი სურათის ზომისა, რაც ძალიან დახვეწილ ვიზუალს ქმნის.
